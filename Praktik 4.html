Kumpulan Tanya Jawab

Pertanyaan 1: Apa itu HTML?

Jawaban: HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web.

Pertanyaan 2: Apa itu CSS?

Jawaban: CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendefinisikan gaya dan tata letak dari dokumen HTML.

Pertanyaan 3: Apa itu JavaScript?

Jawaban: JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan fungsionalitas dinamis pada halaman web.

Pertanyaan 4: Bagaimana cara menambahkan gambar ke halaman web?

Jawaban: Anda dapat menambahkan gambar ke halaman web menggunakan elemen `` dengan atribut `src` yang menunjuk ke file gambar.

Pertanyaan 5: Apa itu Responsive Web Design?

Jawaban: Responsive Web Design adalah pendekatan dalam pengembangan web yang memastikan bahwa halaman web dapat ditampilkan dengan baik di berbagai perangkat dan ukuran layar.
